- 1️⃣ What is the difference between var, let, and const?
        -->var: Function-scoped, can be redeclared, old style.
        -->let: Block-scoped, cannot be redeclared, can reassign.
        -->const: Block-scoped, cannot redeclare or reassign, for constants.

- 2️⃣ What is the spread operator (...)?
 ---->Expands elements of arrays or objects.


- 3️⃣ What is the difference between map(), filter(), and forEach()?
    ---> map()
        Transforms each element and returns a new array.
        Original array is unchanged.

    --->filter()
          Selects elements based on a condition and returns a new array.

    --->forEach()
          Iterates over array elements.
          Does not return a new array, mainly for side-effects like console logs or updating DOM.
